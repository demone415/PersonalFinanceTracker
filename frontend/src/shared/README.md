# shared

Reusable code with no business logic, usable by every layer (FSD `shared` layer).

- `ui/` — design-system primitives (shadcn/ui components)
- `lib/` — framework-agnostic helpers (e.g. `cn`)
- `api/` — HTTP client and base query helpers
- `config/` — env access and app-wide constants
- `hooks/` — generic React hooks
